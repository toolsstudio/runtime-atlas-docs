# Alerts Tab
**Runtime Atlas v1.2.0**

---

## Purpose

Aggregates diagnostic alerts generated by the Camera and Audio nodes. Alerts persist for the session until explicitly dismissed.

---

## Alert Severity Levels

| Severity | Meaning |
|----------|---------|
| Info | Noteworthy condition, not a problem |
| Warning | Potential issue that may cause unexpected behaviour |
| Critical | Condition that will cause incorrect or broken behaviour |

---

## Controls

| Control | Action |
|---------|--------|
| Info / Warning / Critical toggles | Filter visible alerts by severity |
| **×** on a row | Dismiss that alert |
| **Clear All** | Dismiss all currently visible alerts |
| **Clear Dismissed** | Permanently remove dismissed alerts from the log |

---

## Alert Row Fields

| Field | Description |
|-------|-------------|
| Severity badge | Info / Warning / Critical |
| Message | Alert description |
| Count | Number of times this message was generated since last cleared |
| Source | Which module generated the alert (Camera, Audio, etc.) |
| Dismiss button | **×** |

---

## Alert Sources

Alerts are generated by modules that implement `IAlertSource`. In v1.2.0:

- `CameraInspectorNode` — camera depth conflicts, missing culling mask layers
- `AudioInspectorNode` — listener count issues, sources with no clip

Custom alert sources can be added by implementing `IAlertSource`. See [Custom Alert Source](../../guides/custom-alert-source.md).
