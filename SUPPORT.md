# Support

**Runtime Atlas v1.2.0** | Publisher: Tools Studio

---

## Before Reporting an Issue

Check [Troubleshooting](troubleshooting.md) first. It covers the most common installation, configuration, and runtime problems with step-by-step resolutions.

When reporting, include:

- Unity version
- Runtime Atlas version (visible in the About tab)
- Operating system
- Description of the issue
- Relevant Console output
