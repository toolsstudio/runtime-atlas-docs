## Community Support

**Discord:** [Join Discord](https://discord.gg/g5sNBZHZBd)

Post in the `#runtime-atlas` channel. Include:

- Unity version
- Runtime Atlas version (shown in the **About** tab)
- Operating system
- Description of the issue
- Relevant Console output (if applicable)
